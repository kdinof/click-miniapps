## Флоу (переходы)

```
Таб «Для разработчиков»
  S1 (токен не сгенерирован)
    └─[клик «Сгенерировать»]─> всплывает ТОСТ (S2) + страница → S3
  S3 (токен сгенерирован, появился блок запуска Mini-App с кнопкой «Начать»)
    └─[клик «Начать»]─> модал конфигурации тестового мини-аппа (S4):
                         ввод субдомена + несколько телефонов
        └─[клик «Запустить миниап»]─> модал успеха со ссылкой на мини-апп (S5)
  S6 (после настройки теста — на главной блок «где доступен / кому доступен»)
    └─[клик «Редактировать»]─> назад в S4 (редактирование: субдомен + телефоны)

  Карточка «Нет приложения?»
    └─[клик «Посмотреть варианты»]─> модал с вариантами запуска (S10)

Таб «Подписание договора»
  S7 (ввод номера телефона, кнопка «Отправить»)
    └─[клик «Отправить»]─> S8 (состояние после отправки телефона)

Таб «Общая информация»
  S9 (форма с инпутами — общая информация о мини-аппе)
```
